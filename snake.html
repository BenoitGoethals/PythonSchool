reset()[source]

Delete the turtle’s drawings and restore its default values.

No argument.

Delete the turtle’s drawings from the screen, re-center the turtle and set variables to the default values.

Example (for a Turtle instance named turtle): >>> turtle.position() (0.00,-22.00) >>> turtle.heading() 100.0 >>> turtle.reset() >>> turtle.position() (0.00,0.00) >>> turtle.heading() 0.0
